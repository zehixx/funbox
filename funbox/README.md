# Funbox 🎮

Webová aplikace inspirovaná neal.fun - sbírka zábavných miniher a interaktivních experimentů.

## 🚀 Hry a funkce

- **⏳ Škála času** - Od mrknutí oka po stáří vesmíru
- **🍳 Hrozný šéfkuchař** - Mixuj ingredience a zjisti, co vznikne
- **🪐 Gravitační hřiště** - Fyzikální simulace s kuličkami
- **🌐 Sčítání lidu na internetu** - Statistiky v reálném čase
- **⚡ Reakční test** - Otestuj své reflexy
- **🧠 Pexeso** - Klasická paměťová hra
- **🎨 Hádej barvu** - Poznej barvu podle HEX kódu
- **⌨️ Rychlost psaní** - Změř si WPM

## 🛠️ Technologie

- HTML5
- CSS3 (s CSS variables)
- Vanilla JavaScript
- [Matter.js](https://brm.io/matter-js/) pro fyzikální simulaci

## 📁 Struktura projektu

```
funbox/
├── index.html          # Hlavní HTML soubor
├── css/
│   └── style.css       # Styly
├── js/
│   └── script.js       # JavaScript logika
├── assets/             # Obrázky a další assety
└── README.md           # Tento soubor
```

## 🚀 Jak spustit

1. Naklonuj repozitář:
```bash
git clone https://github.com/tvoje-username/funbox.git
```

2. Otevři `index.html` v prohlížeči, nebo použij lokální server:
```bash
cd funbox
python -m http.server 8000
```

3. Otevři http://localhost:8000

## 📝 Poznámky

- Aplikace je plně responzivní
- Data v "Sčítání lidu" jsou simulovaná na základě globálních statistik
- Gravitační hřiště používá Matter.js pro fyzikální simulaci
